## 시작하기
### 사전 요구사항

이 프로젝트는 `Vuetify`와 `Vue.js`를 기반으로 하고 있습니다. 


### 설치

1. Firebase에서 API 키 발급 [https://firebase.google.com/docs/firestore](https://firebase.google.com/docs/firestore)
2. 레포 클론하기
   ```sh
   git clone https://github.com/Ebee1205/DungDong.git
   ```
3. NPM 패키지 설치
   ```sh
   npm install
   ```
